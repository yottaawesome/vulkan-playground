# Vulkan playground

## Intro

Messing around with the Vulkan API. Currently going through the tutorials.

## Building

You'll need Visual Studio 2026 with the _Desktop development with C++_ workload installed. You'll also need the [Vulkan SDK](https://vulkan.lunarg.com/sdk/home) installed.
